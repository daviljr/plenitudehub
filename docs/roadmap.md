PlenitudeHub Roadmap (summary)

Phase 0 - Pilot (0-4 weeks)
- Deploy api-gateway, modules-service, admin static, landing
- Seed modules (plenitude orientador, discovery, experience, active, finance)
- Pilot partners for RealEstateX (3 partners)
- Organic content (3 reels/posts)

Phase 1 - Growth (1-3 months)
- Add user accounts, persistence in Postgres
- Partner portal, lead payments
- WhatsApp Business integration
- Journeys and AI personalization (ArchAI)

Phase 2 - Scale (3-12 months)
- Internationalization
- K8s, CI/CD, infra as code
- Monetization scale: subscriptions, lead fees, B2B sales
